# データ構造のチュートリアル

時間エントリのリストを含む CSV ファイルを開きます。 これらの時間エントリは、複数のユーザーによって特定の日にログ記録された分です。 目標は、この情報を取得して、各ユーザーが毎日ログ記録した合計時間を時間単位で示す新しい CSV を作成することです。
