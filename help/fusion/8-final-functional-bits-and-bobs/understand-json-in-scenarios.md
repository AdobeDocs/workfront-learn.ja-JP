# シナリオでの JSON について

JSON（JavaScript Object Notation）は、データの保存と交換に使われる一般的な構文で、Workfront Fusion で使用できます。

このビデオでは、次のことを学習します。

* JSON の概要および構造
* JSON の読み取り方法と、シナリオ内で使用するために JSON をデータに変換する方法
* JSON を作成、変更、操作する際の JSON モジュールの使用方法
