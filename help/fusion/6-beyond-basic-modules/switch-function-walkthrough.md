# switch 関数のチュートリアル

単純なデータ変更の場合は、スイッチ関数を使用して、モジュールフィールド内の値を別の値に変換します。この演習では、2 文字のキーを、メールで送信されたプロジェクトの進捗ステータスの実際の名前に変更します。
