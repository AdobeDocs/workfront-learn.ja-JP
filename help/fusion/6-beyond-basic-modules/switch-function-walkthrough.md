# 切り替え機能のチュートリアル

## 概要

単純なデータ変更の場合は、 Switch 関数を使用して、モジュールフィールド内の値を別の値に変換します。 この練習では、2 文字のキーを、電子メールで送信されたプロジェクトの進捗状況ステータスの実際の名前に変更します。
