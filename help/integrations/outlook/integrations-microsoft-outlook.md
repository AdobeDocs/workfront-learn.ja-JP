* Web ベースバージョンのとの統合 [!DNL Outlook] クラウドでホストされている [!DNL Office 365] または [!DNL Outlook Live]. すべてのイベントを [!DNL Outlook] カレンダーと、誕生日や休日のカレンダーなど、選択した関連するカレンダーを [!DNL Workfront] [!UICONTROL ホームカレンダー].
