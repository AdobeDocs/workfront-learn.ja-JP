# マイルストーン：コースの概要

このコースは、プロジェクトマネージャー、プランナー、および [!DNL Workfront] でプロジェクトを管理するその他のユーザーを対象としています。このコースを始める前に、プランナーの基本プログラムを完了するか、[!DNL Workfront] でプロジェクトを作成する際の強力な基盤を持つことをお勧めします。
