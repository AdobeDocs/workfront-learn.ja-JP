# タスクの収益とコストのデフォルトの設定

収益タイプおよびコストタイプを使用する場所と、システムのデフォルトの設定方法について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3457685/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **コストと収益データの表示：** タスクのコストと収益の詳細にアクセスするには、タスクを開き、タスクの詳細を選択し、財務セクションにスクロールします。 &#x200B;
* **タスクのデフォルト設定：** システム管理者は、「設定/プロジェクト環境設定/タスクと問題」に移動して、プロジェクトマネージャーのデフォルトのコストと収益タイプを設定できます。 &#x200B;
* **Common Revenue Type Default:** 「Role Hourly」は、収益タイプの一般的なデフォルトであり、ユーザーが収益率の担当業務を持っている場合、ログ時間を請求可能な可能性があるとして記録することができます。 &#x200B;
* **共通コストタイプ デフォルト：** 「ユーザー時間単位」は、多くの場合、デフォルトのコストタイプとして使用され、効率的なトラッキングのためにユーザーのプロファイルからコストデータを直接取得します。 &#x200B;
