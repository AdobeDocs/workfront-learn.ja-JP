# 顧客プロジェクトの財務の更新

このビデオでは、クライアントプロジェクトの時間、費用の追跡、請求記録の作成に関する詳細なガイドを提供します。 &#x200B;要なポイントは次のとおりです。

**時間と費用のトラッキング：**
プロジェクトの作業時間を追跡し、顧客に請求するためのプロジェクト費用を追跡できます。 &#x200B;
ユーザーのプロファイルや担当業務で、コストや請求率が設定されていることを確認します。 &#x200B;

**タスク設定：**
タスクに適切なコストと収益タイプを割り当てる： &#x200B;
コストタイプには、固定時間単位、ユーザー時間単位（ユーザープロファイルに基づく）、役割時間単位（タスク責任者の担当業務に基づく）が含まれます。

**費用の管理：**
「費用」タブを使用して、プロジェクトレベルの費用を作成します。 &#x200B;
プロジェクトやタスクに費用を割り当て、請求可能または返金可能としてマークします。 &#x200B;

**請求記録：**
請求可能なアイテム（時間、費用、収益）を追跡するために、プロジェクトレベルで請求記録を作成します。 &#x200B;
請求可能な時間と費用を請求記録に追加します。
請求先顧客の後の請求レコードのステータスと請求日を更新します。 &#x200B;

**最終メモ：**
Workfrontは請求可能な商品を追跡しますが、実際の請求は処理しません。ユーザーは独自の請求システムを使用する必要があります。 &#x200B;

このビデオでは、クライアントの請求プロセスを合理化するための適切なセットアップと追跡の重要性を強調しています。 &#x200B;

## 重要な留意点


* **コストと請求率の設定：** ユーザーまたは担当業務のプロファイルでコストと請求率が定義されていることを確認して、プロジェクトの経費を正確に追跡します。 &#x200B;
* **タスクにコストと収益タイプを割り当てる：** タスクには、請求可能な金額を正しく計算するために、適切なコストと収益タイプ（時間単位、ユーザー時間単位、役割単位など）が必要です。 &#x200B;
* **プロジェクト費用の管理：** 「費用」タブを使用して、費用を請求可能または払い戻し可能として作成および分類し、正確な追跡を確保します。 &#x200B;
* **請求記録の作成と更新：**&#x200B;請求記録は、請求可能な時間、費用、収益の追跡に役立ちます。 &#x200B;請求後のレコードに項目を追加し、ステータスを更新し、請求日を記録します。 &#x200B;
* **外部請求システムを使用：** Workfrontは請求可能な項目を追跡しますが、実際の請求は処理しません。ユーザーはクライアントへの請求に独自の請求システムを利用する必要があります。 &#x200B;
