# 担当業務とレートカードの作成

このビデオでは、担当業務とレートカードの作成方法、およびプロジェクトからレートカードを添付および削除する方法について説明します。


>[!VIDEO](https://video.tv.adobe.com/v/3486517/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **担当業務の構造プロジェクトタスク：**&#x200B;担当業務は、タスクに必要なスキルを表し、ユーザーの能力と可用性に基づいてユーザーに割り当てることができます。
* **レート カードの請求率の定義：** レート カードは、担当業務の請求率を一覧表示し、プロジェクトの請求用に顧客との契約として機能します。
* **柔軟なレート管理：**&#x200B;担当業務とレートカードの両方で、有効な日付を使用してコストと請求レートを設定でき、時間ベースの価格変更に対応します。
* **レートカードの上書き：** レートカードがプロジェクトに添付されている場合、そのレートは担当業務のレートを上書きしますが、プロジェクト固有の上書きは影響を受けません。
* **簡単な添付と更新：** レートカードはプロジェクトに添付または削除でき、更新は既存のプロジェクトにプッシュして正確な請求を行うことができます。
