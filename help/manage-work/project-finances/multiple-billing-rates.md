# 担当業務の請求率のカスタマイズ

このビデオでは、プロジェクト内の担当業務の請求レートを管理およびカスタマイズする方法について説明します。 &#x200B;同じ役職に対して複数の請求率を設定し、正確な時間ベースの調整を確保する柔軟性に重点を置いています。 


>[!VIDEO](https://video.tv.adobe.com/v/3457652/?quality=12&learn=on&enablevpops=1)

## 重要な留意点


* **既定の請求レート：** システム管理者は、担当業務ごとに既定の請求レートを設定しますが、個々のプロジェクトに合わせてカスタマイズできます。 &#x200B;
* **請求レートの柔軟性：** プロジェクト内の同じ担当業務に対して複数の請求レートを作成し、カスタマイズされた調整を行うことができます。 &#x200B;
* **時間固有の料金：**&#x200B;開始日と終了日を追加することで、特定の期間に請求レートを設定し、プロジェクトのさまざまなフェーズで正確な料金を適用できます。 &#x200B;
* **簡単なカスタマイズ：**&#x200B;請求レートの変更は簡単です。担当業務を選択し、新しいレートを入力し、「レート」タブから直接更新を保存します。 &#x200B;
* **レートカードの添付：**&#x200B;誰かがグループ用にレートカードを用意し、プロジェクトを進めている会社の場合、レートカードを簡単に添付できます。 添付すると、レートカード内の請求レートはどれも変更できませんが、プロジェクト内で上書きすることはできます。
