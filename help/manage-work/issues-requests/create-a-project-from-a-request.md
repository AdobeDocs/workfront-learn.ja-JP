# イシューまたはリクエストをプロジェクトに変換する

ビデオでは、イシューまたはリクエストをプロジェクトに変換する方法について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/340343/?quality=12&learn=on&enablevpops=1)

## 主な要点

* **変換プロセス：**&#x200B;イシューまたはリクエストをプロジェクトに変換するには、イシューを選択し、3 つのドットのメニューを使用して、必要に応じてテンプレートを使用して変換オプションを選択します。
* **テンプレートの選択：** テンプレートの使用は、一貫性と使いやすさを高めるために推奨されます。&#x200B; プロジェクトの要件に基づいて、適切なテンプレートをレビューして選択できます。&#x200B;
* **プロジェクトの詳細：** プロジェクト名と説明は、デフォルトでイシューからコピーされますが、編集できます。&#x200B; イシューのカスタムフォームもプロジェクトに引き継がれます。&#x200B;
* **問題解決：**&#x200B;元の問題をプロジェクトに関連付け、プロジェクトが完了するまで解決済みとしてマークされないようにすることができます。&#x200B;または、変換時に問題を削除することもでき&#x200B;す。
* **アクセスと表示：** プロジェクトへのアクセス権を依頼者に付与できます。&#x200B;変換されたプロジェクトは、「変換済み」列に表示され、タスクまたはプロジェクトに変換されたかどうかを示しま&#x200B;。
