# イシューまたはリクエストをプロジェクトに変換する

ビデオでは、イシューまたはリクエストをプロジェクトに変換する方法について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/340343/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **変換プロセス：**&#x200B;問題またはリクエストをプロジェクトに変換するには、問題を選択し、3点メニューを使用して、変換オプション（できればテンプレートを使用）を選択します。 &#x200B;
* **テンプレートの選択：** テンプレートの使用は、一貫性と使いやすさを高めるために推奨されます。 &#x200B; プロジェクトの要件に基づいて、適切なテンプレートをレビューし、選択できます。 &#x200B;
* **プロジェクトの詳細：** プロジェクト名と説明は、デフォルトでイシューからコピーされますが、編集できます。 &#x200B; イシューのカスタムフォームもプロジェクトに引き継がれます。 &#x200B;
* **問題解決：**&#x200B;元の問題をプロジェクトに関連付け、プロジェクトが完了するまで解決済みとしてマークされないようにすることができます。 &#x200B;または、変換時に問題を削除することもできます。 &#x200B;
* **アクセスと表示：** プロジェクトへのアクセス権を依頼者に付与できます。 &#x200B;変換されたプロジェクトは、「変換済み」列に表示され、タスクまたはプロジェクトに変換されたかどうかを示します。 &#x200B;
