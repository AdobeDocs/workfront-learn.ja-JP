# リクエストを効率的に検索および管理

このビデオでは、Workfrontでリクエストを検索および管理する方法を説明します。 &#x200B;主に以下の4つの領域で構成されています。

* Workfront ホーム
* リクエストエリア
* チームページ
* カスタムレポートとダッシュボード


>[!VIDEO](https://video.tv.adobe.com/v/3422686/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **Workfront ホームでリクエストを検索：** ユーザーは、役割に応じて、自分に割り当てられたリクエストまたは承認待ちのリクエストを表示できます。 &#x200B;
* **リクエスト領域でのフィルターの使用：** 「自分のリクエスト」や「自分の未解決のリクエスト」などのフィルターを使用すると、ユーザーは自分で作成したリクエストを確認できます。 &#x200B;
* **チームページでのリクエストの管理：** ユーザーは、ワークロードバランサーを使用して、現在のワークロードに基づいてチームメンバーの割り当てを管理することで、未割り当てのチームリクエストを確認し、自分自身または他のユーザーに割り当てることができます。 &#x200B;
* **カスタムレポートとダッシュボード：** ユーザーは、リクエストを追跡するためのカスタムレポートを作成できます。これにより、パーソナライズされた命名、フィルター、ビュー、グループ化が可能になり、リクエストの効果的な管理と優先順位付けが可能になります。 &#x200B;
* **フィルターの「マイ」について：**&#x200B;ホームの「マイ」という用語は、自分に割り当てられたイシューを指します。 リクエストエリアの「my」という用語は、私がリクエストした問題を指します。 &#x200B;
