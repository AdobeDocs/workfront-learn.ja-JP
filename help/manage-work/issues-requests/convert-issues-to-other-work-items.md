# イシューまたはリクエストをタスクに変換する

ビデオでは、イシューまたはリクエストをタスクに変換する方法について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3427605/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **簡単な変換プロセス：** イシュー/リクエストをタスクに変換するには、イシューを開き、3点メニューをクリックして、「タスクに変換」を選択します。 &#x200B; タスク名と説明はデフォルトでコピーされますが、編集できます。 
* **宛先プロジェクトの選択：** タスクの宛先プロジェクト（イシューが存在するデフォルトプロジェクト、またはアクセス可能な別のプロジェクト）を選択できます。 &#x200B;
* **フォームの自動添付：**&#x200B;問題にリンクされたカスタムフォームが自動的にタスクに追加され、すべてのフィールド値がシームレスに引き継がれます。 &#x200B;
* **変換オプション：**&#x200B;主なオプションは次のとおりです。
   * 元のイシューをタスクと共にアクティブに保ちます。 &#x200B;
   * 依頼者がタスクにアクセスできるようにします。 &#x200B;
   * タスクのイシューの予定完了日を保持します。 &#x200B;
* **リンクされたステータスの更新：**&#x200B;元のイシューが保持されている場合、そのステータスはタスクの進行状況に基づいて自動的に更新され、イシューとタスクの同期が確保されます。 &#x200B;


>[!NOTE]
>
>イシューは「予定外作業」を表すので、プロジェクトタイムラインに追加することはできません。 プロジェクトタイムラインは、タスクを意味する「予定作業」のためのものです。
