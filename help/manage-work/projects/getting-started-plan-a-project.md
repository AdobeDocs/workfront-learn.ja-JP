# Workfrontによる効果的なプロジェクト計画

このビデオでは、Workfrontでプロジェクトを効果的に計画する方法に関するガイダンスを提供します。 &#x200B; プロジェクトを予定どおりに完了し、成功させるために、強固な基盤を構築することの重要性を強調しています。 &#x200B;要なステップは、タスク、割り当て、期間、予定時間、先行者の5つの重要なフィールドを把握することです。 &#x200B;れらのフィールドを使用すると、プロジェクト計画を迅速かつ正確に調整できます。 &#x200B;

この動画では、プロジェクトを成功に導くための強固な基盤を構築するために、現実的なタイムライン、チームの稼働状況、正確な計画の重要性が強調されています。 &#x200B;

>[!VIDEO](https://video.tv.adobe.com/v/335086/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **5つの重要なフィールドに集中：** タスク、割り当て、期間、予定時間、先行者を理解して、プロジェクト計画を効率的かつ正確に変更します。 &#x200B;
* **プロジェクトの状態を「計画」に設定：**&#x200B;計画を調整する際に不要な通知が表示されないように、プロジェクトの状態が「計画」に設定されていることを確認します。 &#x200B;
* **適切なビューを使用：**&#x200B;標準ビューまたはカスタマイズされたビューを使用して、タスク、期間、割り当てなどの重要なプロジェクトの詳細に焦点を当てます。 &#x200B;
* **現実的な開始日を選択：** チームの稼働状況に合わせて、休日や週末などの非現実的なシナリオを回避する実用的な開始日を設定します。 &#x200B;
* **スケジューリング モードを活用：** 「開始日」モードを使用して完了日を計算するか、「完了日」モードを使用して開始日を計算し、タイムラインがプロジェクト要件に一致するようにします。 &#x200B;
