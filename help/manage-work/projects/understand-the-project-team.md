# プロジェクトチームについて

このビデオでは、プロジェクトチームが使用できる内容とその管理方法について説明します。

>[!VIDEO](https://video.tv.adobe.com/v/3427227/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **チームの自動入力：** ユーザーがタスクを割り当てられるか、プロジェクト所有者として指定されると、プロジェクトチームに自動的に入力されます。 &#x200B;ただし、ユーザーは割り当てが削除されたときに自動的に削除されるわけではありません。これは手動で行う必要があります。 
* **手動更新：**&#x200B;割り当てがない場合でも、プロジェクトチームのメンバーを手動で追加または削除することで、チームリストが正確かつ最新の状態に保たれます。 &#x200B;
* **コミュニケーションの一元管理：** 「メンバー」タブでは、プロジェクトレベルのすべてのチームメンバーに更新情報を送信できるため、チーム全体のコミュニケーションに便利なツールです。 &#x200B;
* **メール通知：** システム管理者は、追加されたドキュメント、完了したマイルストーンタスク、問題に関するアラートなど、プロジェクトチームのメール通知をアクティブ化して、チームメンバーに常に情報を提供することができます。 &#x200B;
* **プロジェクトの関与の追跡：** Workfrontは、プロジェクト チームを使用して、ユーザーがどのプロジェクトに関与しているかを特定し、ユーザーの関与によるプロジェクトのフィルタリングなどの機能を有効にして、ナビゲーションを簡単にします。 &#x200B;
