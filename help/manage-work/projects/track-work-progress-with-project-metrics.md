# プロジェクト指標を使用した作業の進捗の追跡

このビデオでは、プロジェクト指標を使用してプロジェクトの進捗状況を追跡する方法について説明します。 &#x200B; プロジェクトの左側のパネルにある「指標」タブにアクセスする方法を示します。これには、スクロールして「詳細を表示」をクリックする必要があります。 上部のKPI領域には、完了、未完了、期限切れのタスクなどの主要業績評価指標が表示されます。 &#x200B; ユーザーは指標をクリックして特定のタスクを表示し、チャートを使用してステータスや優先度ごとにタスクや問題をグループ化することができます。 &#x200B;列の上にカーソルを合わせると数量が表示され、チャートには完了と未完了のタスクまたは問題の比率が表示されます。 &#x200B;指標では、担当者のステータス（完了、未完了、今後の予定、期限切れ）にもとづいてタスクを表示することもできます。 &#x200B;指標を表示するための環境設定は、今後の訪問のために保存されます。 &#x200B;最後に、「書き出し」ボタンを使用して、指標ページをPNG ファイルとして書き出す方法をビデオで説明します。 &#x200B;


>[!VIDEO](https://video.tv.adobe.com/v/336667/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **プロジェクト指標へのアクセス：** プロジェクトの左側のパネルにある「指標」タブに移動して、進捗状況を追跡します。 &#x200B;下にスクロールして「詳細を表示」をクリックして検索する必要がある場合があります。 &#x200B;
* **主要業績評価指標（KPI）:** KPI エリアには、完了、未完了、期限切れのタスクが表示されます。 &#x200B; インジケーターをクリックすると、特定のタスクを表示できます。 &#x200B;
* **タスクとイシューのグラフ：**&#x200B;は、タスクまたはイシューをステータスまたは優先度でグループ化し、完了と未完了の比率を示します。 &#x200B;列の上にカーソルを合わせると、量が表示されます。 &#x200B;
* **担当者ごとの指標：** ステータス（完了、未完了、今後の予定、期限切れ）に基づいて、担当者ごとのタスクを表示します。 &#x200B;後の訪問時に設定した設定が保存されます。 &#x200B;
* **指標の書き出し：**&#x200B;画面の右上隅にある「書き出し」ボタンをクリックして、指標ページをPNG ファイルとして書き出します。 &#x200B;
