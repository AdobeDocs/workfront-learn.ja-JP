# プロジェクト管理のフィルター、ビュー、グループ化について詳しく見る

このビデオでは、Workfrontのフィルター、ビュー、グループ化の概要を説明し、プロジェクト、タスク、問題を効率的に管理および分析するのに役立ちます。 &#x200B;要なポイントは次のとおりです。

* **プロジェクトフィルター：** フィルターは、プロジェクトリストを絞り込むようにカスタマイズできます。 &#x200B;例には、「自分のプロジェクト」、「自分が進行中のプロジェクト」、「現在のプロジェクト」、「予定より遅れている」などがあります。 「リスクがある」や「予定より遅れている」などの&#x200B;ィルターについて詳しく説明し、異なるフィールドに依存する方法を示します。
* **プロジェクトビュー：** ビューには、計画、予測、実際の開始日/完了日など、様々なプロジェクトの詳細が表示されます。 &#x200B;の例として、条件の更新と要約を提供する「標準ビュー」、「すべての日付ビュー」、「実物大ストーリービュー」などがあります。 &#x200B;
* **プロジェクトのグループ化：** プロジェクトは、進捗状況や完了日などのフィールドで月ごとにグループ化できるため、より優れた整理と分析が可能になります。 &#x200B;
* **タスクフィルター：** フィルターは、特定の役割に関連する未割り当てのタスクまたはタスクを特定するのに役立ちます。 &#x200B; フィルターをクリアすると、親タスクの構造が表示され、並べ替えはタスク番号で調整できます。 &#x200B;
* **タスクビュー：**「ステータスビュー」などのビューには、進捗、コメント、ドキュメント、イシュー、承認、クリティカルパス、マイルストーンのタスクステータスとフラグが表示されます。
* **タスクのグループ化：** プロジェクトのグループ化と同様に、タスクは進行状況などのフィールドでグループ化できます。 &#x200B;
* **問題：**&#x200B;問題のフィルター、ビュー、グループ化は、プロジェクトとタスクのフィルターと似ています。 &#x200B;人気のビューには「入力情報」や「プライマリ連絡先」などのグループが含まれ、問題を整理するのに役立ちます。 &#x200B;

このビデオでは、カスタマイズを強調し、さらに学ぶためのチュートリアルへのリンクを提供しています。 &#x200B;

## 重要な留意点

* **カスタマイズ可能なフィルター：** 「自分が所有するプロジェクト」や「スケジュールの遅れ」などのフィルターは、プロジェクトリストの絞り込みに役立ちます。ユーザーは、より具体的な結果を得るためにフィルターをカスタマイズしたりスタックしたりできます。 &#x200B;
* **詳細ビュー：「すべての日付ビュー」や「実物物大ストーリービュー」などの** ビューでは、プロジェクトの概要や条件の更新だけでなく、計画、予測、実際の日付に関するインサイトを得ることができます。 &#x200B;
* **効率的なグループ化：** プロジェクトとタスクは、進捗状況や完了日などのフィールドでグループ化できるため、より優れた整理と分析が可能になります。 &#x200B;
* **タスク管理：** タスクのフィルターとビュー（「ステータス表示」など）は、未割り当てのタスクの特定、進捗状況の追跡、コメント、ドキュメント、マイルストーンなどの関連情報へのアクセスに役立ちます。 &#x200B;
* **イシューのトラッキング：** 「入力情報」や「プライマリ連絡先」などのイシューのフィルター、ビュー、グループ化により、イシュー管理を効率化し、送信スケジュールと連絡先を明確にします。 &#x200B;
