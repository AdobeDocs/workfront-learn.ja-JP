# プロジェクトをテンプレートから直接作成

このビデオでは、テンプレートからプロジェクトを作成する方法を説明します。 &#x200B; テンプレートを開く権限がある場合は、テンプレートをショートカットとして使用してプロジェクトを作成できます。 &#x200B;これを行うには、テンプレートを開き、3 ドットメニューにアクセスして「プロジェクトを作成」を選択します。 &#x200B; プロジェクトの詳細を編集できる画面が表示されます。 準備が整&#x200B;たら、「プロジェクトを作成」をクリックして確定します。 

>[!VIDEO](https://video.tv.adobe.com/v/3456013/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **テンプレート アクセス：** テンプレートからプロジェクトを作成するには、テンプレートを開く権限が必要です。 &#x200B;
* **ショートカット オプション：** テンプレートを使用すると、プロジェクトをゼロから開始する場合と比較して、便利なショートカットになります。 &#x200B;
* **3点メニュー：** テンプレートを開き、3点メニューから「プロジェクトを作成」を選択します。 &#x200B;
* **プロジェクトの詳細を編集：** 「プロジェクトを作成」を選択すると、プロジェクトの詳細を編集してから最終処理を行うことができます。 &#x200B;
* **作成を完了：** プロセスを完了する準備ができたら、「プロジェクトを作成」をクリックします。 &#x200B;
