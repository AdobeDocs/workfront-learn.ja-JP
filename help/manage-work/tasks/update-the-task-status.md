# 正確なレポート用にタスクステータスを更新

このビデオは、タスクステータスを更新する方法、理由、頻度に関するチュートリアルを提供します。

>[!VIDEO](https://video.tv.adobe.com/v/3471168/?captions=jpn&quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **タスクの状態を更新する方法を教えてください。** タスクのステータスを更新できる場所はいくつかあります。 最も一般的なものは、プロジェクトページのタスクリストにあります。
* **タスクの状態を更新することが重要なのはなぜですか？** プロジェクトやタスクに関するタイムリーかつ正確なレポートは、担当者によるステータスの更新によって異なります。 一貫した更新がないと、表示が損なわれ、決定が行き詰まり、進捗が軌道に乗らなくなるリスクがあります。
* **どれくらいの頻度でタスクステータスを更新したらよいですか？：**&#x200B;各作業日の終わりに作業中のすべてのタスクのステータスを更新します。 これにより、プロジェクトとタスクのレポートの精度が 1 日以内に維持されます。
