# Workfrontでタスクを効率的に管理

このビデオでは、Workfront でのタスク管理について説明します。 プロジェクト計画内のタスクの編集、削除、追加、複製、再整理など、効率的なタスク管理を強調しています。

>[!VIDEO](https://video.tv.adobe.com/v/335088/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **クイック変更のためのインライン編集：** フィールドをダブルクリックしてタスク情報を直接編集し、Tab キーを使用してフィールド間を移動して更新を高速化します。 &#x200B;
* **ごみ箱バックアップでタスクを削除：**&#x200B;削除されたタスクは、システム管理者のみがアクセスできる30日間、ごみ箱に移動され、必要に応じて回復可能性が確保されます。 &#x200B;
* **親子構造を持つタスクの追加：** 「上に挿入」または「下に挿入」を使用して、選択したタスクの親子構造を継承しながら、特定の場所にタスクを追加します。 &#x200B;
* **効率を高めるためのタスクの複製：** タスクまたはタスクのセットを複製して時間を節約し、すべてのタスクの詳細をコピーして変更や再配置を容易にします。 &#x200B;
* **インデント、インデント、ドラッグ&amp;ドロップを使用したタスクの再編成：** インデント、アウトデント、ドラッグ&amp;ドロップなどの直感的なツールを使用して、タスク階層を調整したり、プロジェクト内でタスクを移動したりできます。 &#x200B;
