# プロジェクトプランからのタスクの割り当て

ビデオでは、プロジェクト計画でタスクを割り当てる様々な方法について説明します。 フィルター、テンプレート、リソース管理機能などのツールを使用して、タスクの割り当てを効率化し、プロジェクトのワークフローの最適化を強調しています。


>[!VIDEO](https://video.tv.adobe.com/v/335092/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **クイック割り当て用のインライン編集：**&#x200B;名前を入力してリストから選択し、タスクを簡単に割り当てることができます。先行入力機能を使用すると、選択を高速化できます。 &#x200B;
* **効率化のための一括編集：** タスクを選択し、一括編集を使用して、リストから名前を選択して、同じユーザーに複数のタスクを割り当てます。 &#x200B;
* **合理化された割り当ての担当業務：**&#x200B;担当業務を使用して、タスクの割り当てを促進します。特に、プロジェクトテンプレートの場合に役立ちます。 &#x200B;担当業務は、リソース管理とキャパシティプランニングのために別々に記録されます。 &#x200B;
* **Collaborationのチーム割り当て：** チーム名を選択してチームにタスクを割り当てます（チームアイコンで表示）。 チームメンバー&#x200B;タスクを自動的に割り当てたり、チームリードを再割り当てしたりできます。ただし、レポート用にチーム名は保持します。 &#x200B;
* **リソース管理の統合：**&#x200B;担当業務とチームの割り当てにより、リソースのキャパシティと可用性の計算がサポートされ、プロジェクトの計画と実行が強化されます。 &#x200B;
