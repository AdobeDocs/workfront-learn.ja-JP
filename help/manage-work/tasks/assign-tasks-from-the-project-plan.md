# プロジェクトプランからのタスクの割り当て

このビデオでは、プロジェクトプランでタスクを割り当てる様々な方法について説明します。 特に、フィルター、テンプレート、リソース管理機能などのツールを使用して、タスクの割り当てを合理化し、プロジェクトワークフローを最適化します。


>[!VIDEO](https://video.tv.adobe.com/v/335092/?quality=12&learn=on&enablevpops)

## 重要ポイント

* **クイック割り当てのインライン編集：** 名前を入力してリストから選択することでタスクを簡単に割り当てることができ、選択を高速化する先行入力機能を備えています。&#x200B;
* **効率のための一括編集：** タスクを選択し、一括編集を使用して、リストから名前を選択することで、複数のタスクを同じユーザーに割り当てます。&#x200B;
* **割り当てを効率化するための担当業務：** 担当業務を使用して、特にプロジェクトテンプレートでのタスクの割り当てを容易にします。&#x200B; 担当業務は、リソース管理とキャパシティ計画のために個別に記録されます。&#x200B;
* **Collaborationのチーム割り当て：** チーム名を選択してチームにタスクを割り当てます。チームアイコンが表示されます。&#x200B; チームメンバーはタスクを自分で割り当てることができ、チームリードはレポートのチーム名を保持したまま、タスクを再割り当てできます。&#x200B;
* **リソース管理の統合：** 担当業務とチーム割り当ては、リソースの処理能力と可用性の計算をサポートし、プロジェクトの計画と実行を強化します。&#x200B;
