# 作業量によるタスク時間の見積もり

ビデオでは、正確な見積もりを決定するのが困難な場合に、Workfront の作業量機能を使用してタスクの予定時間数を見積もる方法について説明します。
作業量、予定時間数、期間タイプなどの関連する列を追加して、タスク管理を効率化するカスタム作業量ビューを作成する方法を示します。

>[!VIDEO](https://video.tv.adobe.com/v/3447400/?captions=jpn&quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **予定時間を見積もるための作業量の機能：** Workfrontの作業量の機能は、タスクの予定時間を期間に基づいて見積もるのに役立ち、予定時間を見積もるのが困難な場合にプロジェクト計画を作成しやすくします。 &#x200B;
* **カスタマイズ可能な作業量の割合：** Workfrontでは、タスク期間の割合を使用して予定時間を計算します（例：Medium作業量= 50%、大規模作業量= 75%）。これらの割合は必要に応じて調整できます。 &#x200B;
* **単純期間タイプ要件：**&#x200B;作業量は、単純期間タイプのタスクでのみ使用できます。 &#x200B;作業量を使用する前に、他の期間タイプを持つタスクを単純に変更する必要があります。 &#x200B;
* **インライン編集および一括編集オプション：**&#x200B;作業量は、インライン編集を使用して個々のタスクに合わせて調整したり、一括編集機能を使用して複数のタスクを同時に調整したりできます。 &#x200B;
* **カスタム作業量ビューの作成：** ユーザーは、カスタム作業量ビューを作成して、作業量の設定を簡単に管理および編集できます。 &#x200B;これには、作業量の列と、予定時間や期間タイプなどのその他の関連フィールドを追加することが含まれます。 &#x200B;


[!UICONTROL 作業量]の詳細については、[作業量の概要](https://experienceleague.adobe.com/docs/workfront/using/manage-work/tasks/task-information/work-effort.html?lang=ja)の記事を参照してください。
