# カレンダーレポートインターフェイスについて

カレンダーレポートは、作業とオフをカレンダービューに表示する動的なレポートです。 次のオブジェクトに関する日付情報をカレンダーレポートに表示できます。

* タスク
* プロジェクト
* 問題
* 休暇

このビデオでは、以下について学習します。

* カレンダーのグループ化
* カレンダービュー
* カレンダーアクションメニュー
* 概要情報
* アドホックカレンダー項目の作成方法
