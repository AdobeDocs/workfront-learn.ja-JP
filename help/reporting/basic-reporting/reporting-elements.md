# Workfrontのレポート機能の詳細

このビデオでは、Workfrontのレポート要素（フィルター、ビュー、グループ化）を使用して、プロジェクト、タスク、その他のオブジェクトのリストレポートの貴重な情報にアクセスして調整する方法を説明します。 &#x200B; フィルターは、アクティブなプロジェクトや割り当てられたタスクのみを表示するなど、特定の条件にもとづいてリストを絞り込むのに役立ちます。 &#x200B; ビューでは、収益や費用などの特定のフィールドを強調表示して、列に表示される情報を決定します。 &#x200B; グループ化は、完了予定日や進捗状況などの基準にもとづいてリストを整理します。

このビデオでは、レポート要素をカスタマイズして他のユーザーと共有する方法も示しています。 &#x200B;れらのツールを活用することで、ユーザーはニーズに合わせてレポートをカスタマイズし、可視性と管理を向上させることができます。

>[!VIDEO](https://video.tv.adobe.com/v/335145/?quality=12&learn=on&enablevpops=1)

## 重要な留意点

* **3つのレポート要素：** Workfrontでは、フィルター、ビュー、グループ化を使用して、プロジェクト、タスク、その他のオブジェクトのレポートを絞り込みます。 &#x200B;
* **特定の条件のフィルター：** フィルターは、アクティブなプロジェクトや割り当てられたタスクを表示するなど、条件に基づいてリストを絞り込みます。 &#x200B;
* **フィールド情報のビュー：**&#x200B;表示される列はビューによって決まります。これにより、ユーザーは収益、費用、進捗状況などの特定のフィールドに集中できます。 &#x200B;
* 組織の&#x200B;**グループ化：** グループ化は、予定完了日や進捗状況などの基準に基づいてリストを整理し、ユーザーがプロジェクトの健全性をすばやく評価できるようにします。 &#x200B;
* **カスタマイズと共有：**&#x200B;ユーザーは、カスタムレポート要素を作成および共有して、リストレポートを特定のニーズに合わせて調整できます。
