# 予定時間および期間表示の把握

Burndown グラフと Project ツリーマップグラフでは、Workfront環境でより正確と見なす内容に応じて、予定時間または期間別に情報を表示できます。

![期間ではなく計画時間を選択する画像](assets/section-1-5.png)

デフォルトは予定時間です。 期間に切り替えた場合、 [!DNL Analytics] Workfrontログインの間や戻ってくる
