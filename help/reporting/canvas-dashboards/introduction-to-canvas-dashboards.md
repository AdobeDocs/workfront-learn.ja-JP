# キャンバスダッシュボードの概要

キャンバスダッシュボード機能は、Workfrontでダッシュボードを作成する別の方法です。 現在は、一般的なBetaになっています。つまり、ベータ版に登録されていれば、すべてのWorkfrontのお客様がこの機能を使用できます。 システム管理者が登録を行う必要があります。 このビデオでは、その方法を説明します。
