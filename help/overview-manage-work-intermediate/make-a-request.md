# リクエストする

多くの組織では、新しいプロジェクトを開始したり、新しい製品を開発したり、成果物を作成したりする最初のステップは、Workfront でリクエストすることです。 リクエストの進行状況を追跡し、必要に応じて追加情報を提供できます。

このビデオでは、次の方法を学習します：

* リクエストエリアに移動する
* リクエストする
* 送信されたリクエストを表示する
* リクエストをコピーする
* リクエストの下書きを検索する
* 最後の 3 つのリクエストパスを確認する
